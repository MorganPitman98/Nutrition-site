Workout Plans

Hyrox Workouts

  • Hyrox Simulation: 1km run + 50 wall balls, 1km run + 50 burpee broad jumps...
  • Hyrox Strength Circuit: Deadlifts, sled pushes, lunges, wall balls
  • Partner Hyrox: Relay-style with alternating run + stations

Running Workouts

  • Interval: 8 x 400m sprints with 90s rest
  • Tempo: 20-minute run at threshold pace
  • Long Run: 8–12km at moderate, sustainable pace

Zone 2 Endurance

  • Zone 2 Treadmill: 45 mins at 65–75% max HR
  • Outdoor Run: 60 mins at conversational pace
  • Zone 2 Cross-Training: 30 min row + 30 min cycle (low effort)
